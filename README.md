# `@DeividasJackus/node-starter`

A Node Starter consisting of:
- Base: ESM + TypeScript
- Formatting: EditorConfig + XO + Prettier
- Frontend: Next.js + TailwindCSS

## Implementation

```sh
git init
npx gitignore node

npm init -y

npm -D i typescript tsx prettier xo
npm i node-fetch

npx tsc --init
npm init xo
```
